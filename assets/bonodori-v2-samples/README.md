# Bon Odori V2 sample assets

These transparent PNG files are design samples and are not wired into the game yet.

- `yagura-two-tier.png`: centered two-tier festival yagura and taiko
- `frog-drummer.png`: happi-clad frog with two drumsticks
- `goldfish-yukata-dancer.png`: red goldfish in a navy yukata
- `koi-yukata-dancer.png`: red-white-black koi in a violet yukata
- `frog-yukata-dancer.png`: green frog in a cream yukata
- `bonodori-layout-sample.png`: complete composition reference using all designs

All PNG files are 1254 x 1254 RGBA images with transparent corners. Keep the individual files separate when implementing the effect so the drummer and dancers can be animated independently.
